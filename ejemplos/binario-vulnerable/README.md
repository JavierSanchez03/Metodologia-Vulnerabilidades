# Binario vulnerable para prácticas

Este directorio contiene un programa vulnerable creado para practicar:

- Análisis estático
- Análisis dinámico
- Fuzzing
- Explotación básica

## Contenido
- `vuln.c`: Código fuente vulnerable.
- Instrucciones de compilación.
- Notas del análisis.
- Capturas relacionadas.

Este binario se utiliza como ejemplo práctico dentro del trabajo del módulo 6.
